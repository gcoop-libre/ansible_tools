## `git-help`

Usage:

```bash

  git-help

```

Generate a _markdown_ output for usage of each command _script_


## `git-ls-remote-heads`

Usage:

```bash

  git-ls-remote-heads

```

List remote branchs of git repository.

Example:

```bash

  git-ls-remote-heads

  master
  develop

```


## `git-ls-remote-tags`

Usage:

```bash

  git-ls-remote-tags

```

List remote tags of git repository.


## `git-projects-list`

Usage:

```bash

  git-projects-list

```

List projects from Gitlab API.

Example:

```bash

  git-projects-list

```
